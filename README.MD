# ZyLib
### A Simple Library to make your coding in lua easier!

#### Documentation

##### Call The Library


```LUA
local ZyLib = loadstring(game:HttpGet("https://raw.githubusercontent.com/zyllusware/ZyLib/main/ZyLib.lua"))()
```
##### Bypass Values

```LUA
ZyLib:BypassValue(NameOfValue,ValueToChange)
```
